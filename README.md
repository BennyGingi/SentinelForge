# SentinelForge

SentinelForge is a Detection Regression Testing Platform for security detections.

## Goal

Given attack telemetry and detection rules, SentinelForge validates whether detections still work after changes.

## Technologies

- C++17
- Python 3
- Docker
- Kubernetes
- Git
